## 👨‍🔬 Sobre o projeto

Este projeto foi desenvolvido por Matheus Luiz Mendes de Souza, aluno do primeiro semestre do curso de Bacharelado em Ciência e Tecnologia da Ilum – Escola de Ciência, instituição de ensino superior criada pelo Centro Nacional de Pesquisa em Energia e Materiais (CNPEM).

Motivado pelo interesse despertado pela disciplina de Termodinâmica, desenvolvi um simulador computacional de gases ideais utilizando Python e Pygame. O projeto foi inspirado na simulação interativa Gas Properties, desenvolvida pelo PhET Colorado.

O objetivo da simulação é permitir que estudantes explorem conceitos fundamentais da Teoria Cinética dos Gases de forma visual e interativa. Através da observação do movimento das partículas e de suas colisões elásticas, é possível compreender como propriedades macroscópicas, como a temperatura, emergem do comportamento coletivo de um grande número de partículas microscópicas.

Além disso, o simulador permite visualizar a distribuição de velocidades das partículas por meio de um histograma, possibilitando uma introdução intuitiva à Distribuição de Maxwell-Boltzmann e ao conceito de equilíbrio térmico.

---

## Funcionalidades

- Simulação do movimento de partículas em um recipiente bidimensional;
- Colisões elásticas entre partículas;
- Colisões elásticas com as paredes do recipiente;
- Controle da temperatura do sistema;
- Controle do número de partículas presentes na simulação;
- Cálculo da temperatura a partir da energia cinética média das partículas;
- Representação visual das velocidades por meio de cores;
- Histograma da distribuição de velocidades das partículas.

## Instalação e execução

### Pré-requisitos

Certifique-se de possuir instalado:

* 🐍 Python 3.10 ou superior
* 🎮 Pygame
* 📓 Jupyter Notebook (caso deseje executar o projeto no notebook)

### Instalação do Pygame

```bash
pip install pygame
```

### Clonando o repositório

```bash
git clone https://github.com/SEU_USUARIO/ideal-gas-lab.git
```

```bash
cd ideal-gas-lab
```

### Executando a simulação

Caso esteja utilizando o script Python:

```bash
python ideal_gas_lab.py
```

Caso esteja utilizando o notebook:

```bash
jupyter notebook
```

Em seguida, abra o arquivo do projeto e execute as células em ordem.

### Controles da simulação

| Tecla | Função                           |
| ----- | -------------------------------- |
| ⬆️    | Aumenta a temperatura do sistema |
| ⬇️    | Diminui a temperatura do sistema |
| ➡️    | Adiciona partículas              |
| ⬅️    | Remove partículas                |
